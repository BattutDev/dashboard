## Description

This API is an Nest.js application using redis for store session's tokens, postgres database for store datas

## Hasing method
This project using sha_1 method for hash passwords

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
